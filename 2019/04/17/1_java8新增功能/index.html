java7，8，9新特性

Veröffentlicht am 2019-04-17

1. java8 —- lambda表达式+stream

lambda其实是一个匿名方法，格式就是：将方法参数和方法体分别写在‘->’的左边和右边，参数列表用‘()’，方法体用‘{}’。

1.1 lambda表达式的一般语法
(Type1 param1, Type2 param2, ..., TypeN paramN) -> {
    statment1;
    statment2;
    //.............
    return statmentM;
}
1.2 单个参数的lambda表达式
param1 -> {  //参数列表可省去'()'
  statment1;
  statment2;
  //.............
  return statmentM;
}
1.3 单个参数，并且单语句的lambda表达式
param1 -> statment //当方法体是单语句时，return、大括号、分号都可以省去

例如： 

int res = param1 -> statment
//相当于
int fun(int param1){
    return statment;
}
int res = fun(param1);

2. 接口可以有静态方法

java8仍要求接口的所有静态成员都必须是公有的。
java9中允许接口中有私有的静态方法，但是静态域和静态成员仍然需要是公有的。
