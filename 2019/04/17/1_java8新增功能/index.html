Posted on 2019-04-17

1. java8 — lambda表达式、stream、方法引用、构造器引用

lambda其实是一个匿名方法，格式就是：将方法参数和方法体分别写在‘->’的左边和右边，参数列表用‘()’，方法体用‘{}’。

此匿名方法是基于函数式接口的，即接口中仅有一个抽象方法，那么这个抽象方法就可以用lambda表达式表示。

1.1 lambda表达式的一般语法
(Type1 param1, Type2 param2, ..., TypeN paramN) -> {
    statment1;
    statment2;
    //.............
    return statmentM;
}
1.2 单个参数的lambda表达式
param1 -> {  //参数列表可省去'()'
  statment1;
  statment2;
  //.............
  return statmentM;
}
1.3 单个参数，并且单语句的lambda表达式
param1 -> statment //当方法体是单语句时，return、大括号、分号都可以省去

例如： 

int res = param1 -> statment
//相当于
int fun(int param1){
    return statment;
}
int res = fun(param1);
1.4 方法引用语法
(args)->objectName.instanceMethod(args)//转换成：
objectName::instanceMethod

(args)->ClassName.staticMethod(args) //转换成：
ClassName::staticMethod

(args)->args.instanceMethod()//转换成：
ClassName::instanceMethod

前两种方式，把lambda表达式的参数当成method的参数。
后一种方式，把lambda表达式的参数当成调用method的对象实例。

1.5 构造器引用

ClassName::new

把lambda表达式的参数当成ClassName构造器的参数

2. Java8 — 接口可以有静态方法

java8仍要求接口的所有静态成员都必须是公有的。
java9中允许接口中有私有的静态方法，但是静态域和静态成员仍然需要是公有的。
